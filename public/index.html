HOW TO USE

1Get API key at console.anthropic.com → API Keys → Create Key
2Paste it top-right — saves automatically
3Pick a genre — including 📱 Mobile/iOS for touch games
4Type an idea, hit ✦ ENHANCE to expand it, then ◈ GENERATE
5Click inside the game viewport to play (WASD + Space)
6Hit ↑ PUBLISH to deploy your game live on Vercel with a shareable URL + Neon leaderboard
7Hit ⬇ EXPORT to get Unity C#, Godot GDScript, HTML5, or Mobile/iOS package
8📚 Library saves every game — click any card to preview live
Publish needs a Vercel token (vercel.com/account/tokens). Leaderboards need a free Neon DB (neon.tech). Unity/Godot exports are game logic scripts — add your own 3D models in-engine.
